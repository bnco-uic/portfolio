## 🚀 Getting Started

### Installation
1. Clone or download the portfolio repository
2. Ensure all files are in the correct directory structure:
   ```
   portfolio/
   ├── index.html
   ├── about.html
   ├── projects.html
   ├── contact.html
   ├── resume.pdf
   ├── assets/
   │   ├── css/
   │   │   └── style.css
   │   ├── js/
   │   │   └── script.js
   │   └── img/
   │       └── profile.jpg
   └── README.md
   ```

### Running Locally
1. Open any HTML file in a web browser
2. Or use a local server:
   - Python: `python -m http.server 8000`
   - Node.js: `npx http-server`
   - Live Server extension in VS Code

### Adding Your Resume
1. Place your PDF resume file in the root directory
2. Name it `resume.pdf`
3. The download link on the contact page will automatically work

### Adding Your Professional Photo
1. Replace `assets/img/profile.jpg` with your professional photo
2. Recommended dimensions: 500px x 500px or square aspect ratio
3. Supported formats: JPG, PNG, WebP

## 🔧 Customization

### Updating Contact Information
Edit the following files with your actual contact details:
- **contact.html**: Phone number, email, location, LinkedIn, GitHub URLs
- **about.html**: Personal information in the about section
- **index.html**: Social media links in footer

### Modifying Content
All content is in HTML files. Simply open and edit:
- Experience details in `projects.html`
- Skills in `about.html`
- Background information in `about.html`
- Contact information in `contact.html`

### Changing Colors
Edit `assets/css/style.css` and modify the CSS variables section at the top:
```css
:root {
  --primary-dark: #1e3a8a;
  --primary-blue: #2c3e50;
  --secondary-blue: #3b82f6;
  --accent-gold: #f59e0b;
  /* ... other variables ... */
}
```
