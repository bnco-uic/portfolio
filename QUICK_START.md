# Quick Start Guide - Bianca's Portfolio Website

## 🎯 What You've Got

A complete, professional portfolio website with 4 main pages:

| Page | File | Purpose |
|------|------|---------|
| Home | `index.html` | Landing page with introduction |
| About | `about.html` | Professional background & skills |
| Experience | `projects.html` | Work experience & projects |
| Contact | `contact.html` | Contact form & resume download |

## ✅ Next Steps

### 1. **Add Your Professional Photo**
   - Replace `assets/img/profile.jpg` with your professional photo
   - Recommended: Square image, 500px x 500px minimum
   - Upload a high-quality headshot

### 2. **Update Contact Information**
   - Edit `contact.html` to verify all contact details
   - Update phone, email, LinkedIn, and GitHub URLs
   - All links have been provided with placeholders

### 3. **Add Your Resume PDF**
   - Place your resume file in the root folder
   - Name it exactly: `resume.pdf`
   - Users can download it from the contact page

### 4. **Customize Content (Optional)**
   - Edit any information to match your preferences
   - All content is in HTML files - easy to modify
   - See `README.md` for detailed customization guide

### 5. **Test Locally**
   - Open `index.html` in a web browser
   - OR use a local server:
     - `python -m http.server 8000`
     - Or use VS Code's Live Server extension

### 6. **Deploy Online**
   - Upload to Netlify (drag & drop deployment)
   - Push to GitHub Pages (free hosting)
   - Or use any web hosting service
